## Environment variables

Set these in .zshenv

```
# Will be used for tmux sessionizer alt k
MAIN_PROJECT="absolute/directory"
# Decide battery to show in polybar
BATTERY_NAME="BAT0"
# Load ssh keys
SSH_KEYS="id_rsa etc"
```
